# Code of Conduct

## Our Pledge

We are committed to maintaining a respectful, inclusive, and constructive project environment. This applies to issues, pull requests, documentation, discussions, and any other project-related interaction.

## Expected Behavior

- Communicate respectfully and professionally.
- Provide feedback that is specific, constructive, and focused on the work.
- Welcome contributors with different levels of experience.
- Ask clarifying questions before assuming intent.
- Respect project scope, maintainers' time, and review decisions.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or insults.
- Publicly sharing private information without consent.
- Repeatedly derailing discussions away from the project topic.
- Any behavior that would reasonably make participation unsafe or hostile.

## Enforcement

Maintainers may remove comments, close discussions, reject contributions, or restrict participation when behavior does not follow this code of conduct.

If you experience or observe unacceptable behavior, contact the repository maintainer privately when possible. Reports should include relevant links, screenshots, or context so they can be reviewed fairly.

## Scope

This code of conduct applies within project spaces and in public spaces when a person is representing the project.
